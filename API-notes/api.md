## API 
### Application Programming Interface
> Em Português fica "Interface de Programação de Aplicativos".
- **Programação de Aplicativos** é criar uma olução que sirva e atenda a um problema de negócio.
- **Interface** é igual a um contrato.

### API é uma aplicação programada de forma padronizada (interface), para servir dados seguindo algum protocolo de comunicação.

**E para quê serve?**

Para receber e processar dados em serviços, para então responder a outras aplicações (seja frontend ou backend).
